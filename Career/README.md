# Machine à candidatures — Mode d'emploi

## Lancer une candidature

1. Copie-colle l'offre d'emploi dans le chat
2. L'agent-carrière analyse l'offre, évalue l'adéquation et rédige une lettre adaptée
3. Vérifie que les pièces sont prêtes dans les dossiers ci-dessous
4. Envoie et mets à jour `Career/Candidatures_envoyees/SUIVI.md`

## Structure

| Dossier | Contenu | Quand l'utiliser |
|---|---|---|
| `CV_master/` | CV maître (toutes versions) | Adapter avant chaque envoi |
| `Lettres/modeles/` | Modèles par type de poste | Base pour chaque lettre |
| `Lettres/envoyees/` | Lettres déjà envoyées | Référence et archive |
| `Realisations/biographies.md` | Toutes les biographies | Copier-coller selon le contexte |
| `Realisations/portfolio-cles.md` | 10 réalisations clés chiffrées | Arguments pour chaque lettre |
| `Recommandations/` | Qui peut recommander, pour quoi | Anticiper avant les envois importants |
| `Enseignement/` | Expériences pédagogiques | Candidatures postes d'enseignement |
| `Direction_artistique/` | Postes de direction détaillés | Candidatures DA / institutions |
| `Financements_obtenus/` | Historique des financements | Argument clé pour toute candidature |
| `Offres/SUIVI.md` | Candidatures en cours | Mettre à jour en temps réel |
| `Candidatures_envoyees/SUIVI.md` | Historique + résultats | Référence et analyse |

## Modèles de lettres disponibles

- `Lettres/modeles/direction-artistique.md` — direction artistique, postes institutionnels
- `Lettres/modeles/poste-enseignement.md` — conservatoire, université, école supérieure

## Règle d'or

Ne jamais envoyer une lettre sans avoir vérifié qu'elle contient **au moins un chiffre** (montant levé, nombre de projets, durée du poste). Les chiffres ancrent la crédibilité.
