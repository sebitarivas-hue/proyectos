# proyectos — hébergement du site public STOPERA

Ce dépôt sert uniquement à **publier le site web public de STOPERA** (dossier `docs/`)
via GitHub Pages. Aucune donnée sensible ici. Les données privées (admin, finances,
dossiers) restent dans le dépôt privé `stopera`.

Site : `docs/` · Mise en ligne : Settings → Pages → Deploy from a branch → `/docs`.
